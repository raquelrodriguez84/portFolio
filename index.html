Proyectos destacados

Creación de base de datos, consultas avanzadas y diseño de un MER

Este proyecto consistió en el diseño y creación de una base de datos relacional, utilizando un Modelo Entidad-Relación (MER) detallado para representar la investigación sobre mariposas capturadas con dos fines específicos, bajo condiciones determinadas.

Características principales:

  • Modelo Entidad-Relación (MER): Representación de la investigación sobre mariposas capturadas con fines específicos y condiciones particulares.
  • Implementación en MySQL: Creación de tablas con claves primarias, foráneas y restricciones de integridad para garantizar la coherencia de los datos.
  • Consultas avanzadas: Utilización de joins, vistas y procedimientos almacenados para realizar consultas eficientes y optimizadas.

Este proyecto ha sido una valiosa oportunidad para aplicar mis conocimientos en el diseño y la implementación de bases de datos relacionales. Utilizando herramientas como MySQL y phpMyAdmin, pude crear una estructura de base de datos sólida y eficiente, aplicando conceptos clave como claves primarias, claves foráneas y restricciones de integridad. Además, he perfeccionado mis habilidades en la elaboración de consultas complejas y optimizadas, utilizando joins, vistas y procedimientos almacenados para garantizar la integridad y eficiencia del sistema.

Tecnologías utilizadas: MySQL, phpMyAdmin y SQL.

Proyectos destacados

Página interactiva con Leaflet, Lightbox, jQuery y AJAX

Este proyecto consiste en una página web interactiva que combina diversas tecnologías y herramientas para ofrecer una experiencia visual y funcional dinámica. Utilizando tecnologías fundamentales como HTML, CSS y JavaScript, el proyecto también hace uso de APIs como Leaflet y OpenStreetMap, además de librerías como Lightbox y técnicas avanzadas como jQuery y AJAX para crear una experiencia fluida y dinámica.

Características principales:

  • Galería de imágenes con Lightbox: Implementación de una galería interactiva que permite visualizar imágenes en pantalla completa, mejorando la interacción del usuario.
  • Mapas interactivos con Leaflet: Integración de mapas dinámicos utilizando la API de Leaflet y datos geográficos de OpenStreetMap, permitiendo una navegación intuitiva.
  • Animaciones fluidas con jQuery: Creación de efectos interactivos y transiciones suaves, mejorando la experiencia del usuario en la interfaz.
  • Integración de datos con AJAX: Implementación de cargas asíncronas de datos para actualizar el contenido de la página sin necesidad de recargarla, optimizando el rendimiento.

Tecnologías utilizadas: HTML5, CSS3, JavaScript, jQuery, Leaflet, Lightbox, OpenStreetMap y AJAX.

Este proyecto ha sido una excelente oportunidad para combinar diversas tecnologías y herramientas de desarrollo web, demostrando mi capacidad para crear aplicaciones interactivas y visualmente atractivas. Además, he aprendido a integrar APIs y librerías de terceros para mejorar la funcionalidad y experiencia del usuario.
